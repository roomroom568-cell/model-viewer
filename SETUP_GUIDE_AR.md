# 🚀 دليل التثبيت والتشغيل السريع

## خطوة 1: تحميل البرامج المطلوبة

### أ) تحميل Node.js (مهم جداً)
👇 **اضغط على الرابط واضغط Download:**
https://nodejs.org/en/download/package-manager

**اختر:**
- إذا كنت على Windows → اضغط Windows Installer
- إذا كنت على Mac → اضغط macOS Installer

---

### ب) تحميل Git
👇 **اضغط على الرابط واضغط Download:**
https://git-scm.com/download/win

---

## خطوة 2: تثبيت البرامج

1. افتح ملف التحميل (Node.js)
2. اضغط **Next** → **Next** → **Install** → **Finish**
3. افعل نفس الشيء مع Git

---

## خطوة 3: نسخ المشروع على جهازك

### على Windows:

1. افتح **Command Prompt** (ابحث عنها في القائمة)
2. انسخ والصق هذا الأمر:

```
git clone https://github.com/roomroom568-cell/model-viewer.git && cd model-viewer && npm install && npm start
```

3. اضغط **Enter** والانتظر...

### على Mac:

1. افتح **Terminal**
2. انسخ والصق الأمر نفسه أعلاه
3. اضغط **Enter**

---

## ✅ تم!

سيفتح المشروع تلقائياً في متصفحك على:
**http://localhost:8080**

---

## ❓ إذا حدثت مشكلة

أخبرني بـ:
- الرسالة الحمراء التي تظهر
- نظام التشغيل (Windows/Mac)

سأساعدك! 😊
